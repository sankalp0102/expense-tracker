# Expense Tracker 💰

A simple CLI-based Expense Tracker built using Node.js.

## 🚀 Features
- Add expenses
- View all expenses
- Delete expenses
- Shows total spending

## 🛠️ Tech Used
- Node.js
- File System (fs module)

## ▶️ How to Run

1. Clone the repository:
git clone https://github.com/sankalp0102/expense-tracker.git

2. Go to folder:
cd expense-tracker

3. Run commands:

Add expense:
node app.js add Food 200

List expenses:
node app.js list

Delete expense:
node app.js delete 1

## 📂 Project Structure
- app.js → main logic
- data.json → stores expenses
